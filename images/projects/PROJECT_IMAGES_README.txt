PROJECT PREVIEW IMAGES
======================

Place your project preview images in this folder with the following names:

Required Image Files:
--------------------
1. portfolio-preview.jpg    - For Portfolio Website
2. flixter-preview.jpg      - For Flixter App
3. simpletweet-preview.jpg  - For SimpleTweet
4. todo-preview.jpg         - For To-Do App
5. openapi-preview.jpg      - For Open API Project

Image Specifications:
--------------------
- Format: JPG, PNG, or GIF
- Recommended size: 600x315px (or any 1.9:1 aspect ratio)
- Maximum file size: Keep under 500KB for faster loading
- The images will be automatically cropped to fit the grid layout

Tips:
-----
- Use screenshots of your project
- GitHub social preview images work great (download from your repos)
- Make sure images are clear and representative of each project
- All images should have similar styling for consistency

Once you add your images, the website will automatically display them!
